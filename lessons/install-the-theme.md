TKTK

```bash
yarn workspace example add gatsby-theme-shared-nav@"*"
```

`gatsby-config.js`:

```js
module.exports = {
  plugins: ['gatsby-theme-shared-nav'],
};
```

Start the site with `ntl dev` and it'll run, but nothing will change (yet).
